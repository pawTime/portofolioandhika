Sistem Web Kasir Sembako 2024

Proyek akademik ini berfokus pada pengembangan aplikasi kasir berbasis web yang dirancang untuk membantu pengelolaan penjualan di toko sembako, mulai dari pencatatan stok, supplier, hingga transaksi penjualan harian.

Ringkasan Proyek

Sistem ini dibangun menggunakan teknologi web standar seperti HTML untuk struktur halaman, CSS, PHP sebagai bahasa backend untuk proses logika sistem, serta MySQL untuk menyimpan data seperti produk, pemasok, dan transaksi penjualan.

Tujuan utama dari sistem ini adalah memberikan solusi digital yang efisien untuk toko sembako skala kecil hingga menengah. Dengan sistem ini, pemilik toko dapat memantau stok barang, mencatat pemasukan dan pengeluaran, serta mengelola data pemasok dengan lebih rapi dan cepat.

Fitur Utama

  • Dashboard Interaktif: Menampilkan ringkasan transaksi, stok, dan informasi penting lainnya secara real-time.
  • Manajemen Supplier: Mencatat dan mengelola data pemasok sembako seperti beras, minyak, gula, dan lainnya.
  • Manajemen Bahan Baku/Stok: Input dan pengelolaan berbagai item sembako beserta jumlah stok yang tersedia.
  • Transaksi Penjualan: Proses penjualan yang mudah, mencakup produk, jumlah, harga, dan total pembayaran.
  • Laporan Stok & Transaksi: Menyediakan laporan stok dan rekap penjualan harian/mingguan/bulanan.
  • Antarmuka Responsif: Desain web menyesuaikan dengan berbagai perangkat, dari desktop hingga smartphone.
  • Login Admin: Sistem autentikasi dengan akses khusus untuk admin toko.

Teknologi yang Digunakan

  • Frontend: HTML5, CSS, JavaScript
  • Backend: PHP
  • Database: MySQL
  • Tools: XAMPP, Visual Studio Code
